# 🛡️ CyberPhishing 


# 🔴 Phishing Attack Simulation

An interactive cybersecurity project that demonstrates how phishing attacks work from both the attacker (Red Team) and defender (Blue Team) perspectives.

This project was built to educate users on real-world phishing techniques, how attackers think, and how security teams detect and respond to threats.

# 🚀 Overview
**Most cyberattacks don’t start with advanced hacking; they start with a phishing email.**

CyberPhishing is an interactive web-based simulator that walks users through:

- How phishing attacks are created
- How victims are targeted
- How attackers steal credentials
- How defenders detect and stop attacks

# 🎯 Features

***🔴 Red Team (Attacker Perspective)***

Step-by-step phishing attack lifecycle:
- Reconnaissance
- Infrastructure setup
- Email crafting
- Delivery
- Exploitation
- Post-exploitation
- Realistic attacker terminal simulation
- Social engineering tactics breakdown

***🔵 Blue Team (Defender Perspective)***

Incident response workflow:
- Detection & triage
- Containment
- Impact analysis
- Remediation
- Indicators of Compromise (IOCs)
- Simulated SIEM logs
- Threat analysis dashboard

***📧 Phishing Simulator***

Interactive inbox experience
Identify phishing emails in real-time

Analyze:
- Suspicious links
- Fake attachments
- Email headers
- “Spot the Red Flags” training module

***🎯 Types of Phishing Covered***

1. Email Phishing
2. Spear Phishing
3. Whaling
4. Smishing (SMS)
5. Vishing (Voice)

***🧠 Learning Modules***

Why phishing works (psychology & social engineering), Common attacker techniques, Defensive best practices, and Security awareness training concepts.

***🛠️ Tech Stack***

HTML5, CSS3, Vanilla JavaScript, Responsive design with interactive components

# 🎓 Purpose

This project was created as a Cybersecurity Bootcamp Final Project to demonstrate:

- Understanding of phishing attack vectors
- Knowledge of security operations (SOC workflows)
- Ability to think like both an attacker and defender
- Frontend development skills applied to cybersecurity education
